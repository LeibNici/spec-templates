# Git Workflow

> Trunk-based 简化版：单 main 分支 + 短生命周期 feature 分支。
> 适合 1~10 人团队 + AI 协助开发场景。本目录覆盖分支模型、Conventional Commits、PR/Merge 策略、Force Push 规则、与 AI 协作约定、Tag/Release、禁止模式与 Quick Reference。
